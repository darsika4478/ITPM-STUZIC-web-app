📚 Stuzic – Study Planner with Mood-Based Music Recommendation

Stuzic is a web application designed to help students plan tasks, schedule study sessions, receive reminders, and listen to music based on their mood while studying.

This project was developed as a group academic project.

🚀 Features
📝 Study Planner
Task creation and management
Deadlines and priority tracking
Task status workflow
Task filtering and search

📅 Study Schedule & Reminders
Study timetable/calendar view
Session booking with conflict detection
Automated reminders before sessions

🎧 Mood & Music Recommendation
Mood input (1–5 scale / emojis)
Form validation and data storage using Firebase Firestore
Mood-to-playlist recommendation logic
Automatic playlist suggestions for study sessions
Mood history tracking
Mood analytics and insights dashboard

🎵 Music Player & Study Session Tracking
Playlist playback during study sessions
Focus session timer (Pomodoro)
Session tracking and feedback

🛠️ Tech Stack
Frontend
React
Tailwind CSS

Backend / Database
Firebase
Cloud Firestore
